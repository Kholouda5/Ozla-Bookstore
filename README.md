# Ozla Bookstore 📚

Ozla Bookstore is a simple online bookstore website built using HTML, CSS, and JavaScript.

## ✨ Features

- Browse books
- Book categories
- Book details
- Shopping cart interface
- Delivery page
- Responsive and user-friendly design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
